# Capability Modes

## Full script-agentic

Use this when the machine has JMAG Designer, a valid license, and script execution works. Codex can generate scripts, ask you to run them, inspect output files, then iterate.

If external Python automation is configured, the agent may be able to launch non-GUI scripts directly.

## Semi-agentic

Use this when JMAG is installed but Codex cannot directly control it, or when you only provide exports. Codex can still generate scripts, analyze CSV results, inspect logs, and produce repeatable run plans.

## Advisory

Use this when there is no local JMAG. The toolkit remains useful for designing studies, documenting workflows, and preparing scripts for another machine.
