# Security

This toolkit is source-only. It must not publish private JMAG projects, license files, customer data, university files, solver databases, or screenshots unless the owner has explicitly approved publication.

Blocked by default:

- `.jproj`, `.jmag`, `.jcf`, `.jdata`, `.jplot`, `.jmesh`, `.jresult`, `.jdc`
- archives and office files such as `.zip`, `.rar`, `.pdf`, `.xlsx`
- folders named `exports`, `results`, `private`, `scratch`, `user-projects`, `licensed-projects`

Before running generated scripts:

- work on copied projects;
- review file paths and output folders;
- avoid destructive operations until the script has been inspected;
- verify licensing terms for automation and batch usage.
